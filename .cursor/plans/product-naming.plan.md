# Plan: Product naming (*IQ family)

## Overview

Agent-first DevOps and monitoring platform under **SynteraIQ**. All agent-focused products use the **Agent*IQ** pattern; platform-level capabilities use **PolicyIQ** (or **AgentPolicyIQ** if agent-only).

## Product family

| Product | Role |
|--------|------|
| **SynteraIQ** | Platform (existing) |
| **AgentDeployIQ** | Deploy / ship agents (CD) |
| **AgentEvalIQ** | Evaluate / test agents |
| **AgentPromptIQ** | Prompt registry for agents |
| **AgentPolicyIQ** | Guardrails / policy for agents |
| **AgentMonitorIQ** | Observability / monitoring for agents |

## Positioning

- **One-line:** "Agent-first DevOps and monitoring platform."
- **Tagline option:** "Deploy, evaluate, and monitor AI agents â€” with guardrails and prompts in one place."

## Naming rules

- Agent-specific capabilities: **Agent** + [Domain] + **IQ** (e.g. AgentDeployIQ).
- Platform or cross-cutting: [Domain]**IQ** (e.g. PolicyIQ if policy applies beyond agents).
- Keep *IQ suffix consistent across the family for recognition and SEO.
