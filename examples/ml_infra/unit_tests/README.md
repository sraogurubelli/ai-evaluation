# Unit testing examples

Single-case, pytest, small-suite, threshold, and offline unit tests. Run from repo root: `PYTHONPATH=. pytest examples/ml_infra/unit_tests/ -v`. Prerequisites: `benchmarks/datasets/`, API if using adapters, `CHAT_BASE_URL` and `CHAT_PLATFORM_AUTH_TOKEN`.
